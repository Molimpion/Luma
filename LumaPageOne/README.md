# 💻 CodeSquad

## Como rodar o projeto

1. Instale as dependências:

```bash
npm install
```

2. Em uma aba no terminal, inicie o json server

```
npm run start:server

```

3. Em outra aba, inicie o front-end

```
   npm run dev
```
